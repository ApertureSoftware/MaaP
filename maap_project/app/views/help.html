Guida per l' utente

1. Registrazione

Il primo passo per poter utilizzare MaaP e' la registrazione.
e' possibile registrasi cliccando sul pulsante Register! presente nella pagina di login.
Per registrarsi e' richiesto di inserire

  • email: inserire l’indirizzo mail con il quale registrarsi.
  • password: inserire una password;
  • password confirmation inserire nuovamente la password del punto precedente;
  • Signup: Se il form contiene dati validi, la pressione del bottone "Signup" confermera' la registrazione

2. Autenticazione

Per poter effettuare il login e' necessario essere registrati.
Il login permette di accedere alla dashboard del sistema e quindi a tutte le funzionalita' offerte da MaaP.
Per autenticarsi e' necessario compilare il form proposto nella pagina di login
Viene richiesto di inserire email, password e di cliccare sul pulsante Login.
Terminata l’autenticazione, l’utente viene reindirizzato sulla dashboard.
Per il recupero della password vedi il capitolo 4.Recupero Password.

3. Logout

Per poter effettuare il logout e' necessario essere registrati ed autenticati.
Il logout permette di terminare la propria sessione di lavoro. Un volta effettuato il logout, al successivo accesso alla piattaforma sara' richiesto di autenticarsi.
Per effettuare il logout cliccare sul pulsante con scritto Logout nella barra di navigazione.

4. Recupero Password

Per poter procedere con un recupero password e' necessario conoscere l’email di registrazione.
Il recupero dalla password avviene partendo dalla pagina di login cliccando sul link Forgot your password?

Verra' mostrato un form dove sara' richiesto di inserire l'email di registrazione e cliccare il pulsante di Get it!

L’utente ricevera' un messaggio nella casella di posta associata contenente la nuova password per l'accesso. Dopo aver recuperato la password e' fortemente consigliato cambiarla nuovamente con una più sicura.

5. Accesso e modifica profilo personale

Per accedere alla pagina di modifica del profilo cliccare sul pulsante "Profile" in alto a sinistra nella barra di navigazione.

La pagina Profile visualizza le informazioni personali dell’utente.
La modifica della password di registrazione avviene cliccando sul pulsante "Edit".

Viene proposto un form nel quale va inserita la nuova password due volte.
Se l’utente desidera interrompere l’operazione di modifica password può cliccare il pulsante Discard.

6. Visualizzazione Dashboard

La Dashboard e' la pagina principale dalla quale e' possibile selezionare una delle collection presenti nel sistema.
Per accedervi e' necessario aver eseguito l’autenticazione.

7. Apertura Collection Index

La Collection Index e' la pagina in cui viene visualizzato il contenuto di una collection in forma tabellare.
Per aprire una collection index e' necessario cliccare sul relativo link presente nel menu' collection oppure nel link visualizzato nella pagina dashboard.

La pagina collection index visualizzera' l’intero contenuto della collection in forma paginata.

8. Apertura della Document View

La Document View di un documento visualizza gli attributi del documento in forma tabellare.
Per accedervi è necessario premere il pulsante "View" posizionato accanto ad ogni documento.

9. Come riportare problemi e anomalie

Per la segnalazione di problemi o suggerimenti è a disposizione il sistema di Issue del repository ufficiale di MaaP a questo indirizzo.
